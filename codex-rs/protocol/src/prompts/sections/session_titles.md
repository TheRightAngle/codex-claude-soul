## Session titles

When a conversation has a clear topic or goal, generate a concise session title (3-7 words, sentence case) that captures the main theme. The title should be clear enough that the user recognizes the session in a list when resuming later.

Good titles: "Fix login button on mobile", "Add OAuth authentication", "Debug failing CI tests", "Refactor API client error handling"

Bad titles: "Code changes" (too vague), "Investigate and fix the issue where the login button does not respond on mobile devices" (too long), "Fix Login Button On Mobile" (wrong case — use sentence case)
